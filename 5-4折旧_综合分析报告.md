## 二、各资产类别详细分析

### 2.1 建筑物折旧

#### 数据特征
- **原值**: 79,543.04万元（来自建设投资，在运营期第1年一次性计入）
- **当期折旧费**: 3,778.29万元/年（从运营期第1年开始，每年恒定）
- **净值**: 逐年递减

#### 计算逻辑
```
折旧年限 = 20年
年折旧额 = 原值 / 折旧年限 = 79,543.04 / 20 = 3,778.29万元

运营期第1年净值 = 原值 - 年折旧额 = 79,543.04 - 3,778.29 = 75,764.75万元
运营期第2年净值 = 运营期第1年净值 - 年折旧额 = 75,764.75 - 3,778.29 = 71,986.45万元
...
```

#### 关键特点
1. 原值在建设期各列（E-G列）为0，在运营期第1年（H列）一次性计入
2. 从运营期第1年开始每年计提折旧
3. 采用直线法折旧

  - 运营期第2-3年（I-J列）: 7,954.30万元/年（30%）
  - 运营期第4-20年（K-AD列）: 0万元
- **剩余待销售资产净值**:
  - 运营期第1年: 23,862.91万元
  - 运营期第2年: 15,908.61万元
  - 运营期第3年: 7,954.30万元
  - 运营期第4-20年: 0万元

#### 计算逻辑
```
摊销期限 = 3年（运营期第1-3年）

运营期第1年摊销 = 成本 × 10% = 26,514.35 × 10% = 2,651.43万元
运营期第2年摊销 = 成本 × 30% = 26,514.35 × 30% = 7,954.30万元
运营期第3年摊销 = 成本 × 30% = 26,514.35 × 30% = 7,954.30万元

累计摊销 = 2,651.43 + 7,954.30 + 7,954.30 = 18,560.03万元（70%）
剩余待摊销 = 26,514.35 - 18,560.03 = 7,954.32万元（30%）→ 期末回收

#### 关键特点
1. 在3年内摊销70%，剩余30%作为期末回收资产
2. 摊销比例: 第1年10%，第2-3年各30%
3. 摊销完成后净值归零

## 四、计算逻辑总结

### 4.1 建筑物折旧计算

**公式**:
```
年折旧额 = 原值 / 折旧年限
净值(t) = 原值 - 年折旧额 × (运营年份 - 建设年份)
```

**计算示例**:
```
年折旧额 = 79,543.04 / 20 = 3,778.29万元

净值计算:
运营期1（第4年）: 79,543.04 - 3,778.29 = 75,764.75万元
运营期2（第5年）: 75,764.75 - 3,778.29 = 71,986.45万元
运营期3（第6年）: 71,986.45 - 3,778.29 = 68,208.15万元
```

### 4.2 机器设备折旧计算

**公式**:
```
年折旧额 = 原值 / 折旧年限
净值(t) = 原值 - 年折旧额 × (运营年份 - 建设年份)
```

**计算示例**:
```
原值 = 0万元
年折旧额 = 0 / 10 = 0万元
净值 = 0万元（所有年份）
```

### 4.3 销售固定资产摊销计算

**公式**:
```
摊销总额 = 销售固定资产成本 × 摊销比例
剩余净值 = 成本 - 累计摊销
```

**摊销规则**:
```
运营期1: 摊销10% = 成本 × 0.10
运营期2: 摊销30% = 成本 × 0.30
运营期3: 摊销30% = 成本 × 0.30
剩余30%: 期末回收，不计入运营期
```

**计算示例**:
```
成本 = 26,514.35万元

运营期1:
  摊销 = 26,514.35 × 0.10 = 2,651.43万元
  累计摊销 = 2,651.43万元
  剩余净值 = 26,514.35 - 2,651.43 = 23,862.92万元

运营期2:
  摊销 = 26,514.35 × 0.30 = 7,954.30万元
  累计摊销 = 2,651.43 + 7,954.30 = 10,605.73万元
  剩余净值 = 26,514.35 - 10,605.73 = 15,908.62万元

运营期3:
  摊销 = 26,514.35 × 0.30 = 7,954.30万元
  累计摊销 = 10,605.73 + 7,954.30 = 18,560.03万元
  剩余净值 = 26,514.35 - 18,560.03 = 7,954.32万元

运营期4-20:
  摊销 = 0万元
  剩余净值 = 0万元（已全部摊销或期末回收）
```

### 4.4 合计计算

**公式**:
```
原值合计 = 建筑物原值 + 机器设备原值 + 销售固定资产成本

当期折旧摊销合计 = 建筑物当期折旧 + 机器设备当期折旧 + 销售资产当期摊销

净值合计 = 建筑物净值 + 机器设备净值 + 销售资产剩余净值
```

**计算示例**:
```
运营期1:
  原值合计 = 79,543.04 + 0 + 26,514.35 = 106,057.38万元
  当期折旧摊销合计 = 3,778.29 + 0 + 2,651.43 = 6,429.72万元
  净值合计 = 75,764.75 + 0 + 23,862.92 = 99,627.67万元

运营期2:
  原值合计 = 106,057.38万元
  当期折旧摊销合计 = 3,778.29 + 0 + 7,954.30 = 11,732.59万元
  净值合计 = 71,986.45 + 0 + 15,908.62 = 87,895.07万元

运营期3:
  原值合计 = 106,057.38万元
  当期折旧摊销合计 = 3,778.29 + 0 + 7,954.30 = 11,732.59万元
  净值合计 = 68,208.15 + 0 + 7,954.32 = 76,162.47万元

运营期4-20:
  原值合计 = 106,057.38万元
  当期折旧摊销合计 = 3,778.29 + 0 + 0 = 3,778.29万元
  净值合计逐年递减

def calculate_depreciation(building_value, equipment_value, sales_assets_value,
                          construction_period, operation_period):
    """
    计算固定资产折旧和摊销

    参数:
        building_value: 建筑物原值
        equipment_value: 机器设备原值
        sales_assets_value: 销售固定资产成本
        construction_period: 建设期（年）
        operation_period: 运营期（年）

    返回:
        包含所有年份折旧和摊销数据的字典
    """
    total_years = construction_period + operation_period
    results = []

    for year in range(1, total_years + 1):
        year_data = {
            "year": year,
            "period_type": "建设期" if year <= construction_period else "运营期",
            "operation_year": year - construction_period if year > construction_period else 0
        }

        # 建筑物折旧
        if year <= construction_period:
            # 建设期
            year_data["building_original_value"] = 0
            year_data["building_depreciation"] = 0
            year_data["building_net_value"] = 0
        elif year == construction_period + 1:
            # 运营期第1年，原值计入
            year_data["building_original_value"] = building_value
            year_data["building_depreciation"] = building_value / 20
            year_data["building_net_value"] = building_value - building_value / 20
        else:
            # 运营期第2年及以后
            op_year = year - construction_period
            accumulated_depreciation = (op_year - 1) * (building_value / 20)
            year_data["building_original_value"] = 0
            year_data["building_depreciation"] = building_value / 20
            year_data["building_net_value"] = building_value - op_year * (building_value / 20)

        # 机器设备折旧
        if year <= construction_period:
            year_data["equipment_original_value"] = 0
            year_data["equipment_depreciation"] = 0
            year_data["equipment_net_value"] = 0
        elif year == construction_period + 1:
            year_data["equipment_original_value"] = equipment_value
            year_data["equipment_depreciation"] = equipment_value / 10
            year_data["equipment_net_value"] = equipment_value - equipment_value / 10
        else:
            op_year = year - construction_period
            year_data["equipment_original_value"] = 0
            year_data["equipment_depreciation"] = equipment_value / 10
            year_data["equipment_net_value"] = equipment_value - op_year * (equipment_value / 10)

        # 销售固定资产摊销
        if year <= construction_period:
            year_data["sales_assets_cost"] = 0
            year_data["sales_assets_amortization"] = 0
            year_data["sales_assets_remaining"] = 0
        elif year == construction_period + 1:
            # 运营期第1年：摊销10%
            year_data["sales_assets_cost"] = sales_assets_value
            year_data["sales_assets_amortization"] = sales_assets_value * 0.10
            year_data["sales_assets_remaining"] = sales_assets_value * 0.90
        elif year == construction_period + 2:
            # 运营期第2年：摊销30%
            year_data["sales_assets_cost"] = 0
            year_data["sales_assets_amortization"] = sales_assets_value * 0.30
            year_data["sales_assets_remaining"] = sales_assets_value * 0.60
        elif year == construction_period + 3:
            # 运营期第3年：摊销30%
            year_data["sales_assets_cost"] = 0
            year_data["sales_assets_amortization"] = sales_assets_value * 0.30
            year_data["sales_assets_remaining"] = sales_assets_value * 0.30
        else:
            # 运营期第4年及以后：摊销完成
            year_data["sales_assets_cost"] = 0
            year_data["sales_assets_amortization"] = 0
            year_data["sales_assets_remaining"] = 0

        # 合计
        if year == construction_period + 1:
            year_data["total_original_value"] = building_value + equipment_value + sales_assets_value
        else:
            year_data["total_original_value"] = 0

        year_data["total_depreciation_amortization"] = (
            year_data["building_depreciation"] +
            year_data["equipment_depreciation"] +
            year_data["sales_assets_amortization"]
        )

        year_data["total_net_value"] = (
            year_data["building_net_value"] +
            year_data["equipment_net_value"] +
            year_data["sales_assets_remaining"]
        )

        results.append(year_data)

    return results
