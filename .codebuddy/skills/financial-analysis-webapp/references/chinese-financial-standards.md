# 建设项目经济评价方法与参数 (第三版) 关键要点

## 基本原则

### 1. 评价周期
- **建设期 (Construction Period)**: 项目从开始建设到建成投产所需时间
- **运营期 (Operation Period)**: 项目从投产到计算期末的时间
- **计算期 (Calculation Period)**: 建设期 + 运营期

### 2. 评价体系
建设项目财务评价包括以下层面：
- **财务盈利能力分析**: 净现值(NPV)、内部收益率(IRR)、投资回收期
- **偿债能力分析**: 借款偿还期、利息备付率、偿债备付率
- **财务生存能力分析**: 累计盈余资金
- **不确定性分析**: 盈亏平衡分析、敏感性分析

## 核心计算表

### 1. 投资估算表
包含以下内容：
- 工程费：建筑工程费、设备购置费、安装工程费
- 工程建设其他费：管理费、技术服务费、配套设施费、土地使用费等
- 预备费：基本预备费、涨价预备费
- 建设期利息
- 流动资金

### 2. 资产形成表
**固定资产**:
- 房屋建筑
- 机械设备
- 其他固定资产

**无形资产**:
- 土地使用权
- 专利权
- 其他无形资产

### 3. 折旧摊销表
**折旧计算** (直线法):
```
年折旧额 = (固定资产原值 - 预计净残值) / 折旧年限
净残值 = 固定资产原值 × 残值率
```

**摊销计算** (直线法):
```
年摊销额 = 无形资产原值 / 摊销年限
```

### 4. 成本费用估算表
**总成本费用构成**:
- 外购原材料费
- 外购燃料及动力费
- 工资及福利费
- 修理费
- 其他费用（制造费、管理费、营业费）
- 折旧费
- 摊销费

### 5. 利润与利润分配表
计算步骤:
```
营业收入
- 营业税金及附加
- 增值税
= 营业收入净额

营业收入净额
- 总成本费用
= 利润总额

利润总额
- 所得税
= 净利润

净利润
- 提取法定盈余公积金
- 提取任意盈余公积金
+ 以前年度未分配利润
- 分配给投资者的利润
= 本年未分配利润
```

### 6. 现金流量表

**项目投资现金流量表** (全部投资):
- 现金流入：营业收入、回收固定资产余值、回收流动资金
- 现金流出：建设投资、流动资金、经营成本、营业税金及附加、所得税
- 净现金流量 = 现金流入 - 现金流出
- 累计净现金流量

**项目资本金现金流量表** (自有资金):
- 现金流入：营业收入、回收固定资产余值、回收流动资金
- 现金流出：项目资本金投入、借款本金偿还、借款利息支付、经营成本、营业税金及附加、所得税
- 净现金流量

**投资各方现金流量表**:
- 现金流入：实分利润、资产处置收益分配
- 现金流出：实缴资本、租赁资产支出
- 净现金流量

### 7. 资产负债表
**资产**:
- 流动资产：货币资金、应收账款、存货
- 在建工程（建设期）
- 固定资产净值
- 无形及其他资产净值

**负债及所有者权益**:
- 流动负债：应付账款
- 建设投资借款
- 所有者权益：项目资本金、资本公积、累计未分配利润

### 8. 财务指标计算

**净现值 (NPV)**:
```
NPV = Σ[Ct / (1 + i)^t]
```
其中: Ct = 第t年净现金流量, i = 折现率

**内部收益率 (IRR)**:
使NPV=0时的折现率

**投资回收期 (Payback Period)**:
累计净现金流量首次为正的年份

## 税率标准

### 增值税
- 建筑税率：9%
- 设备税率：13%
- 服务税率：6%

### 所得税
- 企业所得税税率：25%

### 附加税
- 城市维护建设税：7%
- 教育费附加：5%
- 地方教育费附加：2%

### 土地增值税
四级超率累进税率:
- 增值率 ≤ 50%: 税率30%, 速算扣除系数0
- 50% < 增值率 ≤ 100%: 税率40%, 速算扣除系数5%
- 100% < 增值率 ≤ 200%: 税率50%, 速算扣除系数15%
- 增值率 > 200%: 税率60%, 速算扣除系数35%

```
土地增值税 = 增值额 × 适用税率 - 扣除项目金额 × 速算扣除系数
```

## 借款还本付息

### 还款方式
1. **等额本金**: 每年偿还本金相同，利息逐年减少
2. **等额本息**: 每年偿还金额相同，本金逐年增加

### 计算要点
- 宽限期：建设期通常只付息不还本
- 偿还期：运营期开始还本付息
- 利率：区分长期借款（5年以上）和短期借款（1年及以内）

## 数据精度要求

- 所有金额数据保留2位小数
- 百分比数据保留2位小数
- 率类数据保留4位小数（如IRR、折现率等）

## Excel模板规范

### 工作表命名规范
- 数字前缀表示类别（如1建设投资、5-1材料）
- 中文描述便于识别
- 特殊符号：收入后有空格（"6收入 "）、资本金后有空格（"11资本金现金 "）

### 数据布局
- 首行：标题、单位、序号
- 首列：项目名称、序号
- 年份列：建设期（第1年、第2年...）、运营期（第1年、第2年...）
- 合计列：各项目汇总

### 颜色标记
- 黄色背景：输入数据区域（需要用户填写）
- 白色背景：计算结果区域（自动计算）

## 常见计算公式

### 固定资产折旧
```
年折旧额 = (固定资产原值 - 净残值) / 折旧年限
累计折旧 = Σ年折旧额
净值 = 固定资产原值 - 累计折旧
```

### 无形资产摊销
```
年摊销额 = 无形资产原值 / 摊销年限
累计摊销 = Σ年摊销额
净值 = 无形资产原值 - 累计摊销
```

### 总成本费用
```
总成本费用 = 外购原材料费 + 外购燃料及动力费 + 工资及福利费 + 
           修理费 + 其他费用 + 折旧费 + 摊销费 + 财务费用
```

### 利润总额
```
利润总额 = 营业收入 - 营业税金及附加 - 增值税 - 总成本费用
```

### 净利润
```
净利润 = 利润总额 - 所得税
```

### 投资回收期
```
投资回收期 = (累计净现金流量首次为正的年份 - 1) + 
             (上年累计净现金流量绝对值 / 当年净现金流量)
```
